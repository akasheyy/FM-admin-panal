# Admin UI Kit

A beautiful, modern admin panel with dashboard, gallery management, and settings. Built with React and Tailwind CSS.

## Features

- Modern dashboard with statistics
- Gallery management for images
- Settings page for configuration
- Responsive design
- Built with shadcn/ui components

## Technologies Used

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Installation

1. Clone the repository:
```sh
git clone <YOUR_GIT_URL>
cd <YOUR_PROJECT_NAME>
```

2. Install dependencies:
```sh
npm install
```

3. Start the development server:
```sh
npm run dev
```

## Usage

- Dashboard: View statistics and overview
- Gallery: Manage and view images
- Settings: Configure application settings

## Deployment

Deploy to your preferred hosting platform (Vercel, Netlify, etc.) by building the project:

```sh
npm run build
```

Then upload the `dist` folder to your hosting provider.
